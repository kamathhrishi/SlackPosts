# SlackPosts
Blog application I developed using Nodejs , Express,  Mongoose and MongoDB 

Recommended

Node 8.11 Version 

Other dependencies:

<ul>
  <li>Mongoose</li>
  <li>BodyParser</li>
  <li>express</li>
</ul>

By default runs on 



<h1>Run Application</h1>

```
git clone https://github.com/kamathhrishi/SlackPosts.git
cd SlackPosts
node App.js
```

Go to a Web Browser and run 

```
localhost:3000
```
